# Q12. Difference Between Microcontroller and Microprocessor

| Parameter | Microcontroller (Arduino UNO) | Microprocessor (Raspberry Pi) |
|-----------|-------------------------------|-------------------------------|
| CPU Speed | 16 MHz | 1 GHz or higher |
| On-chip RAM | 2 KB SRAM | Uses external RAM (typically 1 GB or more) |
| On-chip Flash | 32 KB Flash Memory | Uses external storage (MicroSD card) |
| Operating System | No Operating System | Runs Linux or other Operating Systems |
| Typical Use Case | Embedded systems, sensor interfacing, automation | Multimedia, networking, AI, complex IoT applications |
| Power Consumption | Low | Higher |

## Conclusion

A microcontroller is suitable for simple embedded and real-time applications with low power consumption. A microprocessor provides higher processing power and is suitable for advanced applications requiring an operating system and multitasking.
