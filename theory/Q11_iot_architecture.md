# Q11. IoT Architecture

## 4-Layer IoT Architecture

```
+------------------------------------------------------+
|                Application Layer                     |
| Smart Home, Smart Agriculture, Healthcare, Industry  |
+------------------------------------------------------+
|                 Processing Layer                     |
| Cloud Computing, Database, Analytics, IoT Platform   |
+------------------------------------------------------+
|                  Network Layer                       |
| Wi-Fi, Bluetooth, Ethernet, 4G/5G, MQTT              |
+------------------------------------------------------+
|                 Perception Layer                     |
| Sensors, Actuators, RFID, Arduino                    |
+------------------------------------------------------+
```

## Description

### 1. Perception Layer

- Collects data from the physical environment.
- Includes sensors and actuators.

**Example Device:** DHT11 Temperature Sensor

---

### 2. Network Layer

- Transfers data between devices and the internet.
- Uses communication technologies.

**Example Technology:** Wi-Fi

---

### 3. Processing Layer

- Stores, processes, and analyzes collected data.
- Usually consists of cloud servers and databases.

**Example Technology:** Firebase Cloud Database

---

### 4. Application Layer

- Provides services to end users.
- Displays processed information and controls IoT devices.

**Example Application:** Smart Home Mobile App

---

## Summary

| Layer | Example |
|--------|---------|
| Perception Layer | DHT11 Sensor |
| Network Layer | Wi-Fi |
